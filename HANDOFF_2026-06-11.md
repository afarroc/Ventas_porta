# HANDOFF - 2026-06-11 - Ubigeo Fix

## Problema
Los campos departamento/provincia/distrito no cargaban opciones dependientes en el formulario de ventas.

## Solución Implementada
1. **static/data/ubigeo-peru.json** - JSON local con todos los distritos del Perú (2059 registros)
2. **apps/ventas/ubigeo_peru.py** - Regenerado para cargar desde el JSON (fuente única)
3. **static/js/venta-form.js** - Función `initUbigeoPeru()` que:
   - Carga el JSON local vía fetch
   - Carga provincias al cambiar departamento
   - Carga distritos al cambiar provincia
   - Se dispara en DOMContentLoaded y ventaFormLoaded
4. **templates/base.html** - Cache bust agregado al script
5. **templates/ventas/_venta_form_fields.html** - Script ubigeo-config con path del JSON

## Flujo
- Selección de departamento (ej: Lima → value="14")
- fetch → filtra por departamento="14" && distrito="00" && provincia!="00"
- Muestra las 10 provincias de Lima
- Selección de provincia (ej: Lima → value="01")  
- fetch → filtra por departamento="14" && provincia="01" && distrito!="00"
- Muestra los 43 distritos de Lima
